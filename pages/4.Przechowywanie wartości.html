Czym są zmienne?

Zmienne są to swego rodzaju pojemniki, które służą do przechowywania danych. Java posiada różne pudła na różne typy na przykład:
String - przechowuje tekst, wartości są przechowywane w "cudzysłowiu"
int - przechowuje liczby całkowite, np. 42 bądź -96
float - przechowuje liczby wymierne, z cyframi po przecinku, np. 202.8 albo -16.81
char - przechowuje pojedyńczy znak, np 'N' or 't'. Znak znajduje się pomiędzy apostrofami.
boolean - przechowuje wartość pod dwoma postaciami: true lub false

Deklarowanie (Tworzenie) zmiennych

Aby stworzyć zmienną musisz określić jej typ i przypisać jej jakąś wartość:
typ nazwa = wartość;
Gdzie typ jest jednym z typów podanych wyżej, a nazwa jest nazwą tej zmiennej (np. x, y czy z).
Po znaku równa się podajemy wartość którą chcemy przypisać do tej konkretnej zmiennej. Poniżej jest przykład jak stworzyć zmienną przechowującą tekst:
String imie = "Bozydar";
System.out.println(imie);
przykład jak stworzyć zmienną przechowującą liczbę całkowitą:
int wiek = 17;
System.out.println(wiek);
Możesz również stworzyć zmienną bez przypisywania do niej wartości i zrobić to później:
int x;
x = 32;
System.out.println(x);
Weź pod uwagę, że przypisywanie nowej wartości do już istniejącej zmiennej nadpisze jej poprzednią wartość:
int liczba = 10;
liczba = 30;
System.out.println(liczba);
//Wynik w konsoli: 30

Zmienne niezmienne

Jeśli nie chcesz, aby wartość twojej zmiennej została kiedykolwiek nadpisana przez inną wartość musisz dodać tylko słowo final przed typem zmiennej. Zmieni ono tą zmienną w stałą, czyli będzie można ją odczytać, ale nie będzie można jej edytować w żadnym innym miejscu w kodzie niż jej deklaracja:
final int stala = 128;
stala = 64;
// Spowoduje błąd: nie można przypisać nowej wartości do stałej

Inne typy

int mojInt = 128;
float mojFloat = 64.201f;
char mojChar = 'N';
boolean mojBool = true;
String mojString = "Jakis tekst";
Dowiesz się więcej o typach danych w następnym temacie.

Wyświetlanie zmiennych

Metoda println() jest często używana do wypisywania zmiennych. Aby połączyć tekst z zmienną,
należy użyć znaku "+" :
String imie = "Bozydar";
System.out.println("Hello " + imie);
Znaku "+" można także użyć, aby dodać jedną zmienną do drugiej:
String imie = "Bozydar ";
String nazwisko = "Nowak";
String imieINazwisko = imie + nazwisko;
String.out.println(imieINazwisko);
Dla wartości numerycznych, znak "+" działa jako symbol matematyczny (zauważ, że używamy wartości int (integer)):
int x = 5
int y = 6;
String.out.println(x + y);
// Wypisuje wartość x + y
Z powyższego przykładu możemy się spodziewać:

x przechowuje wartość 5
y przechowuje wartość 6
Następnie używamy metody println() do wyświetlenia wartości x + y, która wynosi 11

Deklarowanie kilku zmiennych

Aby zadeklarować więcej niż jedną zmienną tego samego typu, można użyć listy oddzielonej przecinkiem:

Zamiast pisać:
int x = 5;
int y = 6;
int z = 50;
String.out.println(x + y + z);
Mozesz po prostu napisać:
int x = 5, y = 6, z = 50;
String.out.println(x + y + z);

Jedna wartość do kilku zmiennych

Możesz także przypisać tą samą wartość do kilku zmiennych w jednej linijce:
int x, y, z;
x = y = z = 50;
String.out.println(x + y + z);

Identyfikatory

Wszystkie zmienne w Javie muszą być zidentyfikowane za pomocą unikalnych nazw.

Te unikalne nazwy nazywane są identyfikatorami.

Identyfikatory mogą być krótkimi nazwami (jak x i y) lub bardziej opisowymi nazwami (wiek, suma, objetosc).
Zaleca się używanie opisowych nazw, aby stworzyć zrozumiały i łatwy do utrzymania kod:
// Dobrze
int minutyNaGodzine = 60;

// OK, ale nie tak łatwo zrozumieć co tak naprawdę oznacza m
int m = 60;

Ogólne zasady dotyczące nazw zmiennych to:
  • Nazwy mogą zawierać litery, cyfry, podkreślenia i znaki dolara
  • Nazwy muszą zaczynać się od litery
  • Nazwy powinny zaczynać się od małej litery i nie mogą zawierać białych znaków
  • Nazwy mogą zaczynać się od $ i _ (ale nie będziemy tego używać w tym samouczku)
  • Nazwy są rozróżniane wielkością liter ("myVar" i "myvar" to różne zmienne)
  • Zarezerwowane słowa (jak słowa kluczowe Javy, takie jak int lub boolean) nie mogą być używane jako nazwy
