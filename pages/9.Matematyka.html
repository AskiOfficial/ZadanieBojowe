Klasa Math

Klasa Math służy w Javie do wykonywania zarówno podstawowych jak i nieco bardziej skomplikowanych operacji matematycznych.

  • Math.max(x, y)

    Metoda zwraca większą z dwóch liczb.
    Math.max(2,5); // 5

  • Math.min(x, y)

    Metoda zwraca mniejszą z dwóch liczb.
    Math.min(2,5); // 2

  • Math.pow(x, y)

    Metoda zwraca x podniesione do potęgi y.
    Math.pow(2,3); // 8.0

  • Math.sqrt(x)

    Metoda zwraca pierwiastek kwadratowy z liczby x.
    Math.sqrt(16); // 4.0

  • Math.abs(x)

    Metoda zwraca wartość bezwzględną z liczby x.
    Math.abs(-5); // 5

Losowanie liczb

Java posiada wbudowany generator liczb pseudolosowych. Aby wylosować liczbę, należy skorzystać z metody Math.random().
Math.random();
Metoda ta zwraca liczbę z przedziału 0.0 do 1.0.

Aby wylosować liczbę z przedziału 0 do 10, należy skorzystać z poniższego wzoru:
Math.random() * 11;

Jeżeli chcesz wylosować liczbę z przedziału 5 do 15, wystarczy dodać do wyniku 5:
Math.random() * 11 + 5;
