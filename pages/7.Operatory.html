Operatory w Javie

Operatory są znakami, które wykonują działania na zmiennych i wartościach. W Javie jest kilka rodzajów operatorów:
Operator Nazwa Opis Przykład
+ Dodawanie Dodaje dwie wartości 2 + 3 = 5
- Odejmowanie Odejmuje jedną wartość od drugiej 5 - 3 = 2
* Mnożenie Mnoży dwie wartości 2 * 3 = 6
/ Dzielenie Dzieli jedną wartość przez drugą 6 / 3 = 2
% Reszta z dzielenia Zwraca resztę z dzielenia 5 % 3 = 2
++ Inkrementacja Zwiększa wartość o 1 ++x LUB x++
-- Dekrementacja Zmniejsza wartość o 1 --x LUB x--

Przykłady


int a = 5;
int b = 3;
System.out.println(a + b); // Pokaże 8
System.out.println(a - b); // Pokaże 2
System.out.println(a * b); // Pokaże 15
System.out.println(a / b); // Pokaże 1
System.out.println(a % b); // Pokaże 2
System.out.println(a++); // Pokaże 6
System.out.println(--a); // Pokaże 4

Operatory przypisania

Operatory przypisania służą do przypisywania wartości do zmiennych. W Javie jest kilka rodzajów operatorów przypisania:
Operator Przykład Równoważne
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y

Przykłady


int a = 5;
a += 3;
System.out.println(a); // Pokaże 8
a -= 3;
System.out.println(a); // Pokaże 5
a *= 3;
System.out.println(a); // Pokaże 15
a /= 3;
System.out.println(a); // Pokaże 5
a %= 3;
System.out.println(a); // Pokaże 2

Operatory porównania

Operatory porównania służą do porównywania dwóch wartości. Zwraca wartość true albo false. W Javie jest kilka rodzajów operatorów porównania:
Operator Nazwa Opis Przykład
== Równy Porównuje, czy dwie wartości są równe 5 == 3
!= Różny Porównuje, czy dwie wartości są różne 5 != 3
> Większy Porównuje, czy jedna wartość jest większa od drugiej 5 > 3
< Mniejszy Porównuje, czy jedna wartość jest mniejsza od drugiej 5 < 3
>= Większy lub równy Porównuje, czy jedna wartość jest większa lub równa drugiej 5 >= 3
<= Mniejszy lub równy Porównuje, czy jedna wartość jest mniejsza lub równa drugiej 5 <= 3

Przykłady


int a = 5;
int b = 3;
System.out.println(a == b); // Pokaże false
System.out.println(a != b); // Pokaże true
System.out.println(a > b); // Pokaże true
System.out.println(a < b); // Pokaże false
System.out.println(a >= b); // Pokaże true
System.out.println(a <= b); // Pokaże false

Operatory logiczne

Operatory logiczne służą do łączenia wyrażeń logicznych. W Javie jest kilka rodzajów operatorów logicznych:
Operator Nazwa Opis Przykład
&& I Zwraca true, jeśli oba wyrażenia są prawdziwe 5 > 3 && 5 < 3
|| Lub Zwraca true, jeśli jedno z wyrażeń jest prawdziwe 5 > 3 || 5 < 3
! Negacja Zwraca true, jeśli wyrażenie jest fałszywe !(5 > 3)

Przykłady


int a = 5;
int b = 3;
System.out.println(a > b && a < b); // Pokaże false
System.out.println(a > b || a < b); // Pokaże true
System.out.println(!(a > b)); // Pokaże false
