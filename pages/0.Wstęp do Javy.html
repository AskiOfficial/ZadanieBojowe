Wstęp do Javy

Co to Java?


Java jest popularnym językiem programowania, opracowanym przez firmę Oracle w 1995 roku.

Obecnie ponad 3 miliardy urządzeń używa Javy.

Jest ona używana między innymi do:

  • Aplikacji mobilnych (w szczególności na Androida)
  • Aplikacji desktopowych
  • Aplikacji sieciowych
  • Serwery i aplikacje sieciowe
  • Tworzenia gier (najlepszym przykładem jest Minecraft)
  • Połączeń między bazami danych

Dlaczego używać akurat Javy?

  • Java działa na wielu platformach (Windows, Mac, Linux, Raspberry Pi, itd.)
  • jednym z najpopularniejszych języków programowania na świecie.
  • Jave ma duży popyt na obecnym rynku pracy.
  • Łatwe do nauczenia i obsługi.
  • Open-source i darmowe.
  • Jest bezpieczna, szybka i potężna.
  • Ma dużą społeczność i wsparcie (dziesiątki milionów deweloperów)
  • Java jest objektowym językiem programowania, co umozliwia przejżystą strukturę programów oraz pozwala aby kod został wykorzystany ponownie,
    obniżając koszty tworzenia.
  • Ponieważ Java jest podobna do C++ i C#, programistom jest łatwo przejść na Jave i odwrotnie.
