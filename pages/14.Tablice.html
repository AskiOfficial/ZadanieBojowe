Wyobraź sobie segregator w którym są kartki. Każda z kartek jest osobno, ale wszystkie są w segregatorze. Naszym segregatorem będą tablice.
Aby zadeklarować tablicę do typu zmiennej dodajemy nawiasy kwadratowe:
String[] uczniowie;
Właśnie stworzyliśmy miejsce na naszą tablicę która przechowuje zmienne tekstowe.
Teraz przydałoby się, żeby w tym miejscu była faktyczna tablica. Możemy wstawić od razu do niej wartości, albo utworzyć pustą.
Wkładamy do naszej tablicy wartości przez "otworzenie" tablicy nawiasami klamrowymi, a następnie wymiemienie wartości po przecinkach w następujący sposób:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
Lub bez wkładania do niej wartości:
String[] uczniowie = new String[4];
Tablice mają określoną wielkość, więc to co zrobiliśmy w kodzie powyżej to stworzenie pustej tablicy, która ma 4 miejsca na przechowanie wartości.

Odczytywanie elementów z tablicy

Możesz odczytać element tablicy poprzed jego indeks, odwołując się do przykładu z segregatorem - indeks to będzie nasza strona.
Ta linijka kodu pozwoli nam odczytać pierwszy element z tablicy uczniowie:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
System.out.println(uczniowie[0]);
//Wypisze w konsoli: Natalia

Warto zapamiętać

Indeksy zaczynają się od 0

String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
Pierwszy element, czyli "Natalia", ma indeks 0
Drugi element, czyli "Adam", ma indeks 1
I tak dalej...

Tablice mają określony rozmiar

Nasza tablica ma 4 elementy: Natalia, Adam, Maciej, Anna.
Oznacza to, że możesz mieć w niej tylko 4 wartości.
System.out.println(uczniowie[4]);
Ta linijka kodu spowoduje błąd, ponieważ masz tylko na cztery wartości, a próbujesz znaleźć piątą.
To samo stanie się podczas próby przypisania wartości do indeksu 4 bo masz tylko miejsce na te 4 wartości.

Zmiana wartości elementu w tablicy

Żeby zmienić konkretny element musimy odwołać się do jego indeksu:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
uczniowie[0] = "Hania";
System.out.println(uczniowie[0]);
// Teraz zamiast wypisać imię Natalia konsola wyświetli imię Hania

Wielkość tablicy

Możesz sprawdzić ile tablica ma miejsc na wartości kożystająć z właściwości length:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
System.out.println(uczniowie.length);
// Wyświetli wartość 4

Przechodzenie pętlą po tablicy

You can loop through the array elements with the for loop, and use the length property to specify how many times the loop should run. The following example outputs all elements in the cars array:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
for (int i = 0; i < cars.length; i++) { System.out.println(cars[i]); }
Loop Through an Array with For-Each There is also a "for-each" loop, which is used exclusively to loop through elements in arrays:
String[] uczniowie = {"Natalia", "Adam", "Maciej", "Anna"};
for (String i : cars) { System.out.println(i); }
The example above can be read like this: for each String element (called i - as in index) in cars, print out the value of i. If you compare the for loop and for-each loop, you will see that the for-each method is easier to write, it does not require a counter (using the length property), and it is more readable.
