Co to jest pętla?

Pętla to fragment kodu, który wykonuje się wielokrotnie, aż do spełnienia określonego warunku. W języku JavaScript mamy do dyspozycji trzy rodzaje pętli: while, do...while i for. Pętle są bardzo przydatne, gdy chcemy wykonać jakąś operację wielokrotnie, np. wyświetlić wszystkie elementy tablicy, wykonać jakąś operację na każdym elemencie tablicy, itp.

Pętla while

Pętla while wykonuje się, dopóki warunek jest spełniony.
while (warunek)
{
// kod, który się wykona, jeśli warunek jest spełniony
}
Należy pamiętać, żeby w kodzie, który się wykona, zmienić warunek, inaczej pętla będzie wykonywać się w nieskończoność.
int i = 0;
while (i < 5)
{
System.out.println(i);
i++;
}

Pętla do...while

Pętla do...while wykonuje się, dopóki warunek jest spełniony, ale kod wewnątrz pętli wykona się przynajmniej raz.
do
{
// kod, który się wykona, jeśli warunek jest spełniony
}
while (warunek);
int i = 0;
do
{
System.out.println(i);
i++;
}
while (i < 5);

Pętla for

Pętla for jest trochę bardziej skomplikowana, ale za to bardziej elastyczna. Składa się z trzech części:
  • inicjalizacja zmiennej
  • warunek
  • zmiana wartości zmiennej
for (inicjalizacja; warunek; zmiana)
{
// kod, który się wykona, jeśli warunek jest spełniony
}
for (int i = 0; i < 5; i++)
{
System.out.println(i);
}
Kod wewnątrz pętli for wykona się pięć razy, bo zmienna i zaczyna się od zera, a kończy na czterech.
Można także użyć pętli for w środku innej pętli for, co pozwala na tworzenie bardziej skomplikowanych struktur.
for (int i = 0; i < 5; i++)
{
for (int j = 0; j < 5; j++)
{
System.out.println(i + " x " + j + " = " + i * j);
}
}

Pętla for-each

Istnieje jeszcze jeden typ pętli i jest nim pętla for-each. Jest to pętla, która służy do przechodzenia przez wszystkie elementy tablicy. Pętla wykona się tyle razy, ile jest elementów w tablicy.
for (typ zmienna : tablica)
{
// kod, który się wykona, jeśli warunek jest spełniony
}
String[] samoloty = {"Boeing", "Airbus", "Embraer", "Bombardier", "Cessna"};
for (String samolot : samoloty)
{
System.out.println(samolot);
}

Instrukcja break

Przy temacie switchy wspomnieliśmy o instrukcji break. Teraz warto wspomnieć o niej w kontekście pętli. Instrukcja break służy do przerwania pętli. Działa ona w pętlach for, while, do...while oraz for-each.
for (int i = 0; i < 5; i++)
{
if (i == 3)
{
break;
}
System.out.println(i);
}
W tym przypadku pętla for wykona się tylko trzy razy, bo po trzecim wykonaniu pętla zostanie przerwana.

Instrukcja continue

Instrukcja continue służy do przerwania obecnej iteracji pętli i przejścia do kolejnej.
for (int i = 0; i < 5; i++)
{
if (i == 3)
{
continue;
}
System.out.println(i);
}
W tym przypadku pętla for wykona się pięć razy, ale nie wyświetli liczby trzy, bo zostanie przerwana.
