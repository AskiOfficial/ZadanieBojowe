Nici w Javie

Nici, lub łańcuchy znaków (String), są używane do przechowywania tekstu.

Zmienna typu String zawiera zbiór znaków otoczonych podwójnymi cudzysłowami:

String imie = "Bozydar"

Metody łańcuchów znaków

String w Javie jest obiektem i zawiera metody, które mogą wykonywać określone operacje na łańcuchach znaków.

  • Metoda length()

    Metoda length() zwraca długość łańcucha znaków:

    String imie = "Bozydar";
    System.out.println(imie.length()); // Wypisze 7

  • Metody toUpperCase() i toLowerCase()

    Metoda toUpperCase() zwraca łańcuch znaków z dużymi literami, a metoda toLowerCase() zwraca łańcuch znaków z małymi literami:

    String imie = "Bozydar";
    System.out.println(imie.toUpperCase()); // Wypisze BOZYDAR

    System.out.println(imie.toLowerCase()); // Wypisze bozydar

  • Metoda indexOf()

    Metoda indexOf() zwraca indeks pierwszego wystąpienia określonego tekstu w łańcuchu znaków (lub -1, jeśli nie występuje):

    String tekst = "Spróbuj znaleźć gdzie 'wyraz' występuje";
    System.out.println(tekst.indexOf("wyraz")); // Wypisze 23

    Pamiętaj, że Java zaczyna indeksowanie od 0.
    0 jest pierwszym indeksem, 1 jest drugim indeksem, itd.

  • Metoda substring()

    Metoda substring() zwraca podciąg łańcucha znaków, zaczynając od określonej pozycji do końca:

    String tekst = "Przykładowy tekst do zadania";
    System.out.println(tekst.substring(12)); // Wypisze "tekst do zadania"

Konkatenacja nici

Konkatenacja oznacza łączenie dwóch nici w jedną. W Javie używamy operatora + do łączenia nici:

String imie = "Bozydar";
String nazwisko = "Nowak";
String pelneImie = imie + " " + nazwisko;
System.out.println(pelneImie);

Można również użyć metody concat():

String imie = "Bozydar";
String nazwisko = "Nowak";
String pelneImie = imie.concat(" ").concat(nazwisko);
System.out.println(pelneImie);

Dodawanie liczb i łańcuchów znaków

Java używa operatora + do dodawania i konkatenacji.
Liczby są dodawane, a nici są łączone.

Jeżeli spróbujesz dodać liczbę i łańcuch znaków, wynikiem będzie Konkatenacja:
int x = 10;
String y = "20";
System.out.println(x + y); // Wypisze 1020 (jako String)

Znaki modyfikacji

Niektóre znaki w Javie są zarezerwowane i nie mogą być używane w łańcuchach znaków bezpośrednio. Aby użyć tych znaków, musisz użyć znaków modyfikacji:

Znak Opis
\n Nowy wiersz
\r Wiersz powrotu
\t Tabulator
\b Backspace
\' Apostrof
\" Cudzysłów
\\ Ukośnik wsteczny
