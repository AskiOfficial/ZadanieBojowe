Instalacja Javy

Niektóre komputery mogą już posiadać zainstalowana Javę.

Aby sprawdzić czy masz zainstalowaną Javę na komputerze z systemem Windows, wpisz w wierszu poleceń (cmd) następującą komendę:

C:\Users\Nazwa Użytkownika>java -version

Jeżeli Java jest zainstalowana otrzymasz taka lub podobną wiadomość (zależy od wersji):

java version "11.0.1" 2018-10-16 LTS
Java(TM) SE Runtime Environment 18.9 (build 11.0.1+13-LTS)
Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.1+13-LTS, mixed mode)

Jezeli nie masz Javy zainstalowanej, możesz ją pobrać ze strony oracle.com.

!! W tym poradniku, kod pisany jest w edytorze tesktu, lecz można także programować w Zintegrownym Środowisku Programistycznym, takim jak Intellij Community lub Intellij Ultimate.
