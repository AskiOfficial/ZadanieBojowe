Już wiesz, że Java obsługuje zwykłe warunki logiczne z matematyki:
Warunek Opis Przykład
Mniej niż x [jest mniejsze od] y x < y
Mniej lub tyle samo co x [jest mniejsze bądź równe] y x <= y
Więcej niż x [jest większe od] y x > y
Więcej lub tyle samo co x [jest większe bądź równe] y x >= y
Tyle samo x [jest równe] y x == y
Inne niż x [jest różne od] y x != y

Możesz użyć tych warunków, żeby podejmować różne akcje w zależności od wartości. Java posiada następujące wyrażenia warunkowe:
  • if (jeśli) - blok kodu który wykona się jeśli warunek jest spełniony.
  • else (w przeciwnym wypadku) - blok kodu który wykona się, kiedy żaden z powyżej postawionych warunków się nie spełni.
  • else if (w przeciwnym wypadku, jeśli) - blok kodu który wykona się, gdy nowy warunek zostanie spełniony ORAZ pierwszy warunek nie zostanie spełniony
  • switch - określa wiele alternatywnych bloków kodu do wykonania. Więcej o switchu w następnym temacie.

if - jeśli

Użyj warunku jeśli, żeby oznaczyć fragment kodu który wykona się tylko jeśli warunek będzie spełniony.
if(warunek)
{
// blok kodu który wykona się, jeśli warunek się spełni
}
if składa się z samych małych liter. Użycie If/iF/IF spowoduje błąd.
W poniższym przykładzie porównujemy dwie wartości i wypisujemy jakiś tekst:
if(64 > 32)
System.out.println("64 jest wieksze od 32");

//Również w ten sposób możemy porównywać zmienne
int x = 128;
int y = 256;
if(x < y)
{
System.out.println("128 jest mniejsze od 256");
}

Gdzie się podziały nawiasy klamrowe

W pierwszym przykładzie powyżej zniknęły nawiasy obok if. Czy to jest poprawne? W tym przypadku tak.
Jeśli warunkowa jest tylko jedna linijka kodu nie potrzebujesz otwierać specjalnie dla niej klamerek.
if(16 > 8)
System.out.println("16 jest wieksze od 8");
System.out.println("Wykonam sie nawet jesli warunek nie zostanie spelniony");
Każda następna linikja wykona się niezależnie od warunku.

else - w przeciwnym wypadku

Użyj warunku w przeciwnym wypadku, żeby oznaczyć fragment kodu który wykona się tylko jeśli warunek nie zostanie spełniony.
if(warunek)
{
// blok kodu który wykona się, jeśli warunek się spełni
} else
{
// blok kodu który wykona się, jeśli warunek się nie spełni
}

int x = 16;
int y = 8;
if(x > y)
System.out.println("16 jest wieksze od 8");
else
System.out.println("16 nie jest wieksze od 8");
W powyższym przykładzie, widzimy że warunek if nie został spełniony, więc wykonał się blok kodu w else.

Else if - w przeciwnym wypadku, jeśli

Powyższy warunek przyda nam się, gdy chcemy sprawdzić kilka warunków. W przeciwnym wypadku, jeśli, sprawdza kolejne warunki, jeśli poprzednie nie zostały spełnione.
if(warunek1)
{
// blok kodu który wykona się, jeśli warunek się spełni
}
else if(warunek2)
{
// blok kodu który wykona się, jeśli warunek się spełni
}
else
{
// blok kodu który wykona się, jeśli żaden z powyższych warunków się nie spełni
}

int x = 16;
int y = 8;
if(x > y)
System.out.println("16 jest wieksze od 8");
else if(x < y)
System.out.println("16 jest mniejsze od 8");
else
System.out.println("16 jest równe 8");

switch

Switch pozwala na wybór jednej z wielu opcji. W zależności od wyboru, wykona się odpowiedni blok kodu.
switch(wyrażenie)
{
case wartość1:
// blok kodu który wykona się, jeśli wyrażenie jest równe wartość 1
break;

case wartość2:
// blok kodu który wykona się, jeśli wyrażenie jest równe wartość2
break;

default:
// blok kodu który wykona się, jeśli żaden z powyższych warunków się nie spełni
}
Pamiętaj, aby każde case zakończyć break. W przeciwnym wypadku, jeśli warunek zostanie spełniony, wykona się każdy blok kodu poniżej.
int dzien = 4;
switch(dzien)
{
case 1:
System.out.println("Poniedziałek");
break;

case 2:
System.out.println("Wtorek");
break;

case 3:
System.out.println("Środa");
break;

case 4:
System.out.println("Czwartek");
break;

case 5:
System.out.println("Piątek");
break;


Skrócona forma

W Javie, możesz użyć skróconej formy warunków, jeśli blok kodu ma tylko jedną linijkę.
zmienna = (warunek) ? Prawda : Fałsz;

int dzien = 4;
String dzienTekst = (dzien == 6 || dzien == 7) ? "Weekend" : "Dzień roboczy";
System.out.println(dzienTekst);
