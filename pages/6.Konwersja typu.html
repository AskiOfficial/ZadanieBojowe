Konwersja typu w Javie

Konwersja typu to przypisanie wartości jednego typu danych do innego typu.

W Javie, istnieją dwa rodzaje konwersji:
  • Konwersja rozszerzająca (automatyczna) - konwertowanie mniejszego typu na większy typ rozmiaru
    • byte → short → char → int → long → float → double

  • Konwersja zawężająca (ręczna) - konwertowanie większego typu na mniejszy typ rozmiaru
    • double → float → long → int → char → short → byte

Konwersja rozszerzająca

Konwersja rozszerzająca jest wykonywana automatycznie, gdy przekazujemy mniejszy typ danych do większego typu danych:
int mojInt = 9;
double mojDouble = mojInt;
System.out.println(mojInt); // Pokaże 9
System.out.println(mojDouble); // Pokaże 9.0

Konwersja zawężająca

Konwersja zawężająca musi być wykonana ręcznie poprzez umieszczenie typu w nawiasach przed wartością:
double mojDouble = 9.78d;
int mojInt = (int) mojDouble;
System.out.println(mojDouble); // Pokaże 9.78
System.out.println(mojInt); // Pokaże 9
