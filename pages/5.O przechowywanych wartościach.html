Typy danych w Javie

Tak jak wytłumaczono w poprzednim rozdziale, zmienna w Javie musi być określonego typu danych:

int mojInt = 128; // Liczba całkowita
float mojFloat = 64.201f; // Liczba zmiennoprzecinkowa
char mojChar = 'N'; // Znak
boolean mojBool = true; // Zmienna zerojedynkowa
String mojString = "Jakis tekst"; // Tekst

W Javie typy danych są podzielone na dwie grupy:
  • Typy danych podstawowe - byte, short, int, long, float, double, boolean i char
  • Typy danych niestandardowe - takie jak String, Array i Class (o nich dowiesz się w internecie)

Typy danych podstawowych

Typ danych podstawowy określa rozmiar i typ wartości zmiennej, i nie posiada dodatkowych metod.

Jest osiem typów danych podstawowych w Javie:
Typ Rozmiar Przechowywane wartości
byte 1 bajt Całkowite liczby od -128 do 127
short 2 bajty Całkowite liczby od -32,768 do 32,767
int 4 bajty Całkowite liczby od -2,147,483,648 do 2,147,483,647
long 8 bajtów Całkowite liczby od -9,223,372,036,854,775,808 do 9,223,372,036,854,775,807
float 4 bajty Liczby ułamkowe. Wystarczające do przechowywania 6 do 7 cyfr dziesiętnych
double 8 bajtów Liczby ułamkowe. Wystarczające do przechowywania 15 cyfr dziesiętnych
boolean 1 bit Przechowuje wartości true lub false
char 2 bajty Przechowuje pojedynczy znak/literę lub wartości ASCII

Liczby

Typy danych liczbowych są podzielone na dwie grupy:

Typy całkowite przechowują liczby całkowite, dodatnie lub ujemne (takie jak 123 lub -456), bez części dziesiętnej. Dostępne typy to byte, short, int i long. Który typ powinieneś użyć, zależy od wartości liczbowej.

Typy zmiennoprzecinkowe reprezentują liczby z częścią ułamkową, zawierającą jedną lub więcej cyfr dziesiętnych. Są dwa typy: float i double.

Pomimo tego, że w Javie jest wiele typów liczbowych, najczęściej używane dla liczb to int (dla liczb całkowitych) i double (dla liczb zmiennoprzecinkowych).

Typy całkowite

Byte


Typ danych byte może przechowywać liczby całkowite od -128 do 127. Może być używany zamiast int lub innych typów całkowitych, aby zaoszczędzić pamięć, gdy jesteś pewien, że wartość będzie mieściła się w zakresie -128 i 127:

byte mojByte = 100;
System.out.println(mojByte);

Short


Typ danych short może przechowywać liczby całkowite od -32768 do 32767:

short mojShort = 5000;
System.out.println(mojShort);

Int


Typ danych int może przechowywać liczby całkowite od -2147483648 do 2147483647. Ogólnie, i w naszym samouczku, typ danych int jest preferowanym typem danych, gdy tworzymy zmienne z wartością liczbową:

int mojInt = 100000;
System.out.println(mojInt);

Long


Typ danych long może przechowywać liczby całkowite od -9223372036854775808 do 9223372036854775807. Jest używany, gdy int nie jest wystarczająco duży, aby przechować wartość. Zauważ, że wartość jest zakończona "L":

long mojLong = 15000000000L;
System.out.println(mojLong);

Typy zmiennoprzecinkowe

Powinieneś używać typu zmiennoprzecinkowego, gdy potrzebujesz liczby z częścią dziesiętną, taką jak 9.99 lub 3.14515.

Float i double są używane do przechowywania liczb zmiennoprzecinkowych. Zauważ, że wartość musi być zakończona "f" dla float i "d" dla double:

Przykład float:

float mojFloat = 5.75f;
System.out.println(mojFloat);

Przykład double:

double mojDouble = 19.99d;
System.out.println(mojDouble);
Użyć float czy double?

Precyzja wartości zmiennoprzecinkowej określa, ile cyfr wartość może mieć po przecinku. Precyzja float wynosi tylko sześć lub siedem cyfr dziesiętnych, podczas gdy zmienne double mają precyzję około 15 cyfr. Dlatego bezpieczniej jest używać double do większości obliczeń.

Typ boolean

Jest bardzo często używany w programowaniu, będziesz potrzebował typ danych, który posiada tylko dwie wartości, np:

TAK / NIE
ON / OFF
PRAWDA / FAŁSZ

Do tego, Java posiada typ danych boolean, który posiada tylko dwie wartości true lub false:
boolean czyJavaJestFajna = true;
boolean czyNiemieckiJestFajny = false;
System.out.println(czyJavaJestFajna); // Wyświetla true
System.out.println(czyNiemieckiJestFajny); // Wyświetla false

Znaki w Jaive

Znaki


Typ danych char jest używany do przechowywania pojedynczego znaku. Znak musi być otoczony pojedynczymi cudzysłowami, takimi jak 'A' lub 'c':
char mojaKlasa = 'p';
System.out.println(mojaKlasa);
Ewentualnie, jeżeli znasz wartości ASCII, możesz użyć ich do wyświetlania określonych znaków:
char mojaZmienna1 = 65, mojaZmienna2 = 66, mojaZmienna3 = 67;
System.out.println(mojaZmienna1);
System.out.println(mojaZmienna2);
System.out.println(mojaZmienna3);

Tekst


Typ danych string jest używany do przechowywania sekwencji znaków (tekstu). Wartości string muszą być otoczone podwójnymi cudzysłowami:
String powitanie = "Witaj Świecie";
System.out.println(powitanie);
