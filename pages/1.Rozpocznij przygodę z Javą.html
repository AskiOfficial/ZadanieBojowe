Instalacja Javy

Niektóre komputery mogą już posiadać zainstalowana Javę.

Aby sprawdzić czy masz zainstalowaną Javę na komputerze z systemem Windows, wpisz w wierszu poleceń (cmd) następującą komendę:

C:\Users\Nazwa Użytkownika>java -version

Jeżeli Java jest zainstalowana otrzymasz taka lub podobną wiadomość (zależy od wersji):

java version "11.0.1" 2018-10-16 LTS
Java(TM) SE Runtime Environment 18.9 (build 11.0.1+13-LTS)
Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.1+13-LTS, mixed mode)

Jezeli nie masz Javy zainstalowanej, możesz ją pobrać ze strony oracle.com.

!! W tym poradniku, kod pisany jest w edytorze tesktu, lecz można także programować w Zintegrownym Środowisku Programistycznym, takim jak Intellij Community lub Intellij Ultimate.

Instalacja dla Windowsa

Aby zainstalować Jave na Windowsie:
  1. Idź do Właściwości systemu (można je znaleźć w Panel Sterowania > System i zabezpieczenia > System > Zaawansowane ustawienia systemu)
  2. Kliknij na "Zmienne środowiskowe" pod zakładką "Zaawansowane"
  3. Następnie, wybierz zmienną "Path" i Kliknij przycisk edytuj
  4. Kliknij przycisk "dodaj" i wklej ścieżkę, gdzie Java jest zainstalowana i dopisz \bin. Domyślnie, Java jest zainstalowana pod C:\Program Files\Java\jdk-11.0.1 (jeżeli nic nie zmieniałeś podczas instalacji). W tym przypadku, musisz dodać ścieżkę: C:\Program Files\Java\jdk-11.0.1\bin
  5. Na koniec, otwórz Wiersz polecenia (cmd.exe) i wpisz java -version aby zobaczyć czy Java działa na twojej maszynie.

Szybki start

W Javie, każda aplikacja zaczyna się z nazwą klasy i nazwa tej klasy musi pasować nazwie pliku.

Stwórzmy pierwszy plik w Javie, nazwiemy go Main.java, który może być wykonany w każdym edytorze tekstu (np w Notatniku).

Plik powinien zawierać wiadomość "Hello, World", która może być wykonana następującym kodem:

public class Main
{
public static void main(String[] args)
{
System.out.println("Hello, World");
}
}

Nie przejmuj się, jeżeli nie rozumiesz powyższego kodu - wytłumaczymy go w następnych rozdziałach. Narazie, skup się jak uruchomić ten kod.

Zapisz plik pod nazwą "Main.java". otwórz Wiersz polecenia (cmd.exe), nawiguj do katalogu gdzie zapisałeś plik i wpisz "javac Main.java":

C:\Users\Nazwa Użytkownika>javac Main.java

To zkompiluje twój kod. Jeżeli nie ma żadnych błędów, to znaczy, że kod został skompilowany poprawnie. Teraz, możesz uruchomić kod wpisując "java Main":

C:\Users\Nazwa Użytkownika>java Main
Hello, World

Gratulacje! Właśnie uruchomiłeś swój pierwszy program w Javie!
