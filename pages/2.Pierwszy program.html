W poprzednim rozdziale utworzyliśmy plik Javy, który nazwaliśmy Main.java i użyliśmy następującego kodu aby wypisać w konsoli tekst "Hello World":
public class Main
{
public static void main(String[] args)
{
System.out.println("Hello World");
}
}

Wyjaśnienie

Każda linijka kodu która działa w Javie musi być w jakiejkolwiek klasie. W naszym przykładzie nazwaliśmy tą klasę Main Nazwa klasy powinna zawsze zaczynać się dużą literą. Choć nie jest to coś bez czego twój program nie zadziała, to inni będą mogli łatwiej zrozumieć twój kod.

Java jest wrażliwa na wielkość liter!

W konsoli powinien pojawić się tekst:
Hello World

Główna metoda main

Metoda main jest wymagana i zobaczysz ją w każdym jednym programie:
public static void main(String[] args)
Dowolny kod wewnątrz main() się uruchomi, jeśli nie występują błędy. W tym momencie nie martw się słowami kluczowymi. Na razie musisz pamiętać, że każdy program musi mieć klasę i metodę main.

System.out.println()

Wewnątrz metody main() możemy użyć wspomnianego już println(), metody używanej do wypisywania tekstu w konsoli:
public static void main(String[] args)
{
System.out.println("Hello World");
}

Warto zauważyć!

- Nawiasy klamrowe { } zaznaczają początek oraz koniec pewnego bloku kodu.
- Każde polecenie kończy się średnikiem (;), jest jak kropka na końcu każdego zdania.
System jest wbudowaną w Javę klasą zawierającą przydatne elementy. Jest to między innymi out, które oznacza "wyjście" (ang. output).
Metoda println() jest używana, żeby wypisać coś w konsoli, a następnie przejść do następnej linii (lub pliku).
Nie martw się zpecjalnie System, out i println(). Poprostu tak jest, sklejasz je razem i magicznie pojawia się tekst na ekranie.

Wypisywanie tekstu

Nauczyłeś się już, jak używać metody println(), żeby wypisywać tekst w języku Java:
System.out.println("Hello World!");

Cudzysłów

When you are working with text, it must be wrapped inside double quotations marks "". If you forget the double quotes, an error occurs:
System.out.println("This sentence will work!");
System.out.println(This sentence will produce an error);

The Print() Method

There is also a print() method, which is similar to println(). The only difference is that it does not insert a new line at the end of the output:
System.out.print("Hello World! ");
System.out.print("I will print on the same line.");

Print Numbers

You can also use the println() method to print numbers. However, unlike text, we don't put numbers inside double quotes:
System.out.println(3);
System.out.println(358);
System.out.println(2137);
You can also perform mathematical calculations inside the println() method:
System.out.println(3 + 3);
System.out.println(2 * 5);
